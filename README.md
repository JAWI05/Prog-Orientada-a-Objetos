## 📌 Sobre o Repositório
Este repositório contém a implementação de classes e objetos que exploram os conceitos fundamentais da POO em Java. O foco principal é a modelagem de entidades do mundo real e a gestão de estados e comportamentos através de atributos e métodos.

## 💡 Conceitos Aplicados
- **Abstração:** Criação de classes representando entidades do mundo real com atributos e métodos específicos.
- **Encapsulamento:** Uso de modificadores de acesso (`private`) e métodos `getters` e `setters` para proteção de dados.
- **Enumerações (Enums):** Utilização de `enums` para padronização de tipos (ex: Gênero, Curso, Classificação).
- **Construtores:** Inicialização de objetos com parâmetros específicos.
- **Métodos de Comportamento:** Implementação de lógicas específicas como `aumentarSalario()`, `depositar()`, `fazerFogo()`, entre outros.
